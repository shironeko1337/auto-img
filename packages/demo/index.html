Here are several demos using an 800 × 600 image of a predefined focus area.

For all demos below, the image is not stretched, but with a super narrow container, there could be empty space on the border.

Lazy loaded image

Example usage of lazy loaded image with a placeholder.

  • Normally you might want the placeholder to be a base64 image to prevent blank space while loading a placeholder.
  • Placeholder is placed on the center of the container without any distortion.
  • Only <auto-img /> element can be set with a placeholder because this library doesn't control the rendering of a native element. 

<auto-img
  src="https://fastly.picsum.photos/id/200/800/600.jpg?
  hmac=wHrUWgaA9t7IwxTeGzmRrvvnBIMCFvJ29yZVL8dr8EE"
  focus="27,235;452,490"
  padding="10"
  placeholder="YOUR PLACEHOLDER IMAGE"
  defer
  ></auto-img>

  </div>
</div>
